# 产品变更记录 (Changelog)

## [1.0.0] - 2026-02-10

### 变更类型：新增

### 新增功能
- **游戏主界面**：双人对战主界面，包含玩家信息展示、提问记录区、操作输入区
- **游戏设置**：支持玩家名字/头像自定义、出题范围选择、最大提问次数、AI 大模型 API Key 配置
- **AI 智能出题**：根据限定范围自动生成知名人物名字
- **AI 提问判定**：支持"是否提问"和"猜测结果"两种模式的 AI 判定
- **胜负判定与游戏结束**：完整的胜负逻辑、放弃认输机制、游戏结束弹窗
- **重新开始**：清空提问记录、重新出题，保留设置
- **全屏显示**：浏览器全屏模式切换
- **音效系统**：8 种场景音效，支持静音切换
- **动态视觉效果**：10+ 种动画效果，深色霓虹游戏主题

### 设计决策说明
- 采用纯前端架构（HTML + CSS + JS），不引入框架，以降低部署复杂度
- 两位玩家操作相互独立（非回合制），以提升游戏节奏感
- 默认集成智谱 AI，因其中文理解能力优秀且对中国用户友好
- 提问次数耗尽后仅允许猜测，避免游戏无限拖延
- 音效通过 Web Audio API 合成，避免外部依赖
